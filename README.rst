Genscrape API Client
====================

API client lib for Genscrape API


Endpoints
---------

* Scrapers
  
  - get (by scraper_id)
  - all
  - create
  - delete
  - update


* Crawls

  - get (by crawl_id)
  - all
  - start (start crawl for a scraper_id)


* Crawled Items

  - all
  - get_paginated (paginated items by ``page`` and ``per_page``)
  - create
  - delete
